# JEE 2027 Dashboard — Install as App

## Step 1: Upload to GitHub Pages (Free Hosting)

1. Go to https://github.com and sign up / log in
2. Click **New Repository** (top right, + button)
3. Name it: `jee-dashboard` (keep it public)
4. Click **Create Repository**
5. Click **uploading an existing file**
6. Upload ALL files from this folder:
   - index.html
   - manifest.json
   - sw.js
   - icon-192.png
   - icon-512.png
7. Click **Commit changes**

## Step 2: Enable GitHub Pages

1. Go to your repository → **Settings**
2. Scroll to **Pages** (left sidebar)
3. Under Source: select **main** branch → **/root** folder
4. Click **Save**
5. Wait 1-2 minutes → your app is live at:
   `https://YOUR-USERNAME.github.io/jee-dashboard/`

## Step 3: Install on Your Phone

### Android (Chrome):
1. Open the link above in Chrome
2. Tap the **3-dot menu** (top right)
3. Tap **"Add to Home Screen"**
4. Tap **Add** → Done!

### iPhone (Safari):
1. Open the link in Safari
2. Tap the **Share button** (bottom center)
3. Tap **"Add to Home Screen"**
4. Tap **Add** → Done!
